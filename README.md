# SAGE-VLM

SAGE-VLM is a highly efficient Vision-Language Model (VLM) framework designed for real-time autonomous perception on edge devices within complex, dynamic urban environments. It addresses severe multi-camera coordinate aliasing and the erasure of safety-critical spatial topologies caused by generic encodings and destructive pooling through multi-view decoupled 2D rotary positional encoding and a semantic-conditioned visual residual module.
