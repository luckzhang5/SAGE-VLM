SAGE-VLM: Spatially-Decoupled Rotary Encoding and
Semantic Residual Gating for Multi-View Vision-Language Models
Jun Zhang, Bin Zhu, Jiazhan Gao, Zuolong Li,
Jihong Zhu, Member, IEEE, and Askar Hamdulla, Member, IEEE

Abstract

Conceptual overview. SAGE-VLM ensures real-time multimodal perception while natively safeguarding geometric and semantic structures against dilution.

Recent advancements in vision-language models have shown immense potential for autonomous system perception. However, deploying these frameworks in highly dynamic, complex urban environments (e.g., dense traffic intersections, interactive pedestrian zones) remains severely restricted due to the critical demands for real-time inference on edge devices. To address these limitations, we present a highly efficient framework named SAGE-VLM. Existing architectures often suffer from two critical bottlenecks: they apply standard multi-directional 2D encodings that cause severe coordinate aliasing across concatenated multi-view sequences, and they employ destructive text-conditioned pooling mechanisms that collapse rigid spatial topologies, risking the erasure of unprompted safety-critical obstacles.

Within the proposed SAGE-VLM, we introduce two core architectural innovations. First, a Multi-View Decoupled 2D Rotary Positional Encoding (MVD-RoPE) mechanism explicitly injects fine-grained spatial inductive biases natively into attention layers by decoupling continuous view boundaries, eliminating multi-camera spatial confusion. Second, a Semantic-Conditioned Visual Residual (SCVR) module distills an adaptive pure text query to act as a non-destructive semantic gate. Rather than destructively pooling features, this module dynamically filters background noise while using a residual pathway to preserve underlying spatial topology. Remarkably, despite utilizing a highly condensed backbone (20.3M parameters), SAGE-VLM delivers superior cross-modal reasoning and generation accuracy compared to existing baselines in larger parameter regimes.

Keywords: Vision-Language Models, Autonomous Perception, Rotary Positional Encoding, Parameter Efficiency, Cross-Modal Fusion.

Motivation & Challenges

Multi-View Coordinate Aliasing

Standard ID positional encodings or generic multi-directional 2D embeddings fail for concatenated multi-view sequences, causing non-physical coordinate hopping and spatial confusion at camera perspective boundaries.

Destructive Semantic Pooling

Existing text-conditioned hard-dropping or pooling mechanisms destructively compress visual features. If the text fails to prompt a specific object (like a pedestrian), these methods risk "semantic blindness" by erasing critical obstacles.

Severe Edge Constraints

Operating robustly in interactive traffic scenarios demands extremely low-latency inference. Standard LLM-based autonomous models possess billions of parameters, prohibiting real-time execution on resource-constrained edge hardware.

Visual Noise Dilution

High-resolution multi-view observations yield massive token collections. Directly fusing these features without extracting pure semantic anchors often dilutes visual signals with task-irrelevant backgrounds.

Overall Architecture of SAGE-VLM

Comprehensive pipeline overview. SAGE-VLM processes continuous frame sequences through a lightweight backbone, preserves spatial relations via MVD-RoPE, and purifies cross-modal feature flows using the SCVR non-destructive gate before optimal transport aggregation.

SAGE-VLM safely projects high-dimensional multi-view continuous frame sequences into a condensed linguistic representation. The raw visual frames are passed through a frozen, high-efficiency backbone (e.g., MobileViTV2).

To prevent coordinate confusion, features are processed by the MVD-RoPE module. Concurrently, textual tokens are refined via an Adaptive Sparse Extractor into a Pure Text Query. This query dynamically modulates the spatial tokens through the SCVR module via a fail-safe gate, yielding noise-resistant features, which are finally aggregated via a Text-Guided Soft-Sort Pooling (TGSSP) matrix for the downstream language model.

Core Innovations: MVD-RoPE and SCVR

MVD-RoPE: Decouples continuous view boundaries to eliminate cross-view aliasing while explicitly injecting exact 2D topological inductive biases.
Semantic-Conditioned Fusion: Distills an adaptive pure text query to drive a non-destructive residual gate, preserving baseline spatial topologies via an identity bypass (+).

Multi-View Decoupled 2D RoPE (MVD-RoPE): By strictly decoupling the local spatial grids corresponding to independent continuous shots, and applying orthogonal rotary operations across decoupled axes, MVD-RoPE preserves intrinsic 2D geometric distances natively within attention layers at zero additional parameter cost during inference.

Semantic-Conditioned Visual Residual (SCVR): SCVR extracts a purified text query using an adaptive sparse keyword extractor (applying F.relu dynamic sparsity). Instead of destructive hard-dropping, it uses the query to generate a multi-modal residual gate. The residual addition rule ($V_{final} = V_{visual} + Gate \odot V_{visual}$) ensures unprompted critical obstacles are structurally preserved for downstream planning.

Detection Performance & Efficiency

Comparison of model performance on benchmark. Our model surpasses baseline architectures in reasoning capabilities.
Comparison of VRAM utilization and FLOPs. Our Tiny variant operates at just 20.3M parameters and 0.19 GB Memory, highly suitable for edge devices.

Evaluated on the demanding DriveLM benchmark, SAGE-VLM demonstrates a profound pattern of parameter efficiency. Our SAGE-VLM-Tiny configuration (capped at 20.3M parameters) successfully outperforms standard baseline architectures operating with nearly 10x the parameter footprint.

Crucially for edge deployment, SAGE-VLM consumes only 0.19 GB of continuous memory while sustaining 19.90 GFLOPs of processing capability, rendering it exceptionally suitable for real-time local execution in unpredictable urban driving environments.

Conclusion

In this paper, we conducted a systematic investigation into lightweight Vision-Language Models for multi-view autonomous perception and presented SAGE-VLM. By explicitly preserving 2D spatial invariants via MVD-RoPE and preventing semantic blindness to unprompted obstacles via SCVR non-destructive gating, our framework establishes a robust, fail-safe paradigm for highly compressed multi-modal intelligence.

Optimized specifically for real-time edge deployment in complex, dynamic urban driving environments, our 20.3M Tiny model configuration cross-scales and outperforms baseline frameworks utilizing significantly larger parameter regimes across complex multimodal reasoning metrics.
